-----------------------------------------------
	Color Scheme Documentation
-----------------------------------------------


settings
	background 					Sheet - Background color
	foreground 					Sheet - Foreground color	
	caret 						Caret - Foreground color
	invisibles 					Invisible chars - Foreground color
								( tabs, whitespaces, etc. )
								( No effect in at least ST3 )

	lineHighlight 				Line Highlight - Background color
	selection 					Selection - Background color
	selectionBorder 			Selection - Border color
	findHighlight 				Find - Background color of Matches
	findHighlightForeground 	Find - Foreground color of Matches

	gutter 						Gutter - Background color
	gutterForeground 			Gutter - Foreground color
